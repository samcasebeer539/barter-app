# Barter Card Images

This folder contains images for barter cards displayed in the CardWheel component.

## Image Specifications
- **Resolution**: 600 x 850 pixels recommended
- **Format**: PNG or JPEG
- **Max File Size**: 500KB per image (recommended)

## Usage
Images in this folder can be imported and used in the app like this:

```javascript
const cardImage = require('../assets/barter-cards/accept-decline.png');
// or
import acceptDeclineImage from '../assets/barter-cards/accept-decline.png';
```
